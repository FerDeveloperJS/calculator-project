# 📝 Calculator App

Una aplicación web calculadora, para hacer operaciones matemáticas.

## 🚀 Características

- Hacer operaciones básicas como (suma, resta, multiplicación y división). -> En proceso
- Borrado / reset. -> Próximamente

## ⚙️ Instalación

1. Clona el repositorio.
2. Abre el archivo index.html en tu navegador.

## ▶️ Uso 

- Usa los botones para ingresar números y seleccionar la operación matemática deseada.
- Presiona `=` para ver el resultado.

## 🖥️ Tecnologías Utilizadas

- HTML
- CSS
- JavaScript
